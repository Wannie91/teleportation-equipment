Version: 1.0.15
Date: 06.07.2026
  Features:
    - Updated to Factorio 2.1
  Changes:
    - Changed max-teleportation-distance setting from runtime to startup to accurately reflect the range in the prototype
    - Changed required-energy-to-teleport setting to double setting
    - Changed value of required-energy-to teleleport from kW to MJ as the former was too cheap
    - Changed the default value of required-energy-to-teleport from 750kW to 20MJ
    - Changed the default range from 1000 to 500 tiles
    - Added new setting that controls how fast the equipment recharges.
  Optimizations:
    - Changed the energy handling of the equipment. It should now properly recharge and use energy from batteries and reactors.
  Compatibility:
    - Removed a if case which should make the mod independent from quality
